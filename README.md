### General information
CPM is a useful tool for managing your C and C++ projects.

### Building
```
cd build
cmake ..
make
sudo make install
```

### Other information
- The code uses [ANSI Escape Codes](https://gist.github.com/fnky/458719343aabd01cfb17a3a4f7296797) to color text.
